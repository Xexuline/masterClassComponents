None encapuslation inside default encapsulation

The objective of this exercise is to understand the behavior of styles encapsulation when a component with no encapsulation is inside of an object with the default encapsulation

Previus steps:

1- Open the dev tools of your browser
2- Look for the "header" tag in this storybook
3- Watch how the styles are written

Try to answer this questions

1- Why the father component is not red?
2- Why the son component is not yellow?
3- What modifications can you do in the generated HTML to change the color of the son and father component to red? Do these changes editing the HTML of the storybook with the dev tools of the browser
